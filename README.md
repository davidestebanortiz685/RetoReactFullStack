Aplicación web tipo E-commerce desarrollada con React como parte del reto de desarrollo Full Stack.

Demo en vivo
https://davidestebanortiz685.github.io/RetoReactFullStack/

 e-commerce que permite a los usuarios registrarse, iniciar sesión, explorar un catálogo de productos, agregar productos al carrito y realizar un checkout simulado.

FUNCIONALIDADES
Registro e inicio de sesión de usuarios (localStorage)
Galería dinámica de productos
Buscador de productos en tiempo real
Paginación de productos
Carrito de compras con cantidades y total
Previsualización de checkout
Diseño responsivo
